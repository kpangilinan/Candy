Candy Demos:

0.1 and 0.2 Release:

  • Simple Effects - Demonstrates all the effects currently in the Candy library
  • Dynamic Effects - Shows how to change video effects dynamically
  • Brightness Controller - Adjust brightness level while video is playing
  • Dynamic Green Screen - Change green screen background on the fly

0.3 Release:

  • Dynamic RGB Colors - Dynamically manipulate RGB colors.
  • Emboss Effect - A decorative effect designed to "rise" the main part of the image relative to its background.
  • Blur Effect - The visual effect of this blurring technique is a smooth blur resembling that of viewing the image through a translucent screen, distinctly different from the bokeh effect produced by an out-of-focus lens or the shadow of an object under usual illumination.
  • Tilt-Shift Effect - Refers to the use of camera movements on small- and medium-format cameras, and sometimes specifically refers to the use of tilt for selective focus, often for simulating a miniature scene. Sometimes the term is used when the shallow depth of field is simulated with digital postprocessing; the name may derive from the tilt-shift lens normally required when the effect is produced optically.
  • Disperse Effect - Effect that scatters pixels in different directions.
  • Modulate Effect - An effect that varies the brightness of an image's pixels in a periodic way. It can be used add "Venitian blind", "scanline" or "mesh" type effects.
    • Demo #1 - Wavelength: 5, Amplitude: 10, Orientation: h
    • Demo #2 - Wavelength: 20, Amplitude: 30, Orientation: v
  • Pixelate Effect - An effect that transforms an image into a grid of uniformly colored rectangles. So named because it simulates increasing the size of pixels in the image.
    • Demo #1 - 4x4
    • Demo #2 - 8x8
    • Demo #3 - 16x16
  • Pointillize Effect - An effect that transforms an image into a series of uniformly colored dots.
    • Demo #1 - Ellipse, Size: 10, Spacing: 10, Jitter: 0
    • Demo #2 - Ellipse, Size: 10, Spacing: 8, Jitter: 4
    • Demo #3 - Ellipse, Size: 10, Spacing: 6, Jitter: 0
    • Demo #4 - Rectangle, Size: 9, Spacing: 10, xJitter: 0
    • Demo #5 - Rectangle, Size: 10, Spacing: 10, Jitter: 4
    • Demo #6 - Rectangle, Size: 10, Spacing: 8, Jitter: 4
    • Demo #7 - Rectangle, Size: 10, Spacing: 6, Jitter: 0
